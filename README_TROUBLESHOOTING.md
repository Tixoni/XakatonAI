# Решение проблем

## Ошибка: "can't open file 'main.py'"

Эта ошибка обычно возникает из-за проблем с кодировкой пути (кириллица в имени пользователя).

### Решение 1: Используйте run.py
```bash
python run.py
```

### Решение 2: Используйте полный путь
```bash
python "C:\Users\Тихон\Desktop\xakatonAI\main.py"
```

### Решение 3: Переименуйте папку пользователя
Если возможно, переименуйте папку пользователя на латиницу.

## Другие проблемы

### Модель не загружается
- Убедитесь, что есть интернет-соединение (модель скачивается автоматически)
- Проверьте, что указана правильная модель в config.json

### Видео не открывается
- Проверьте путь к файлу в config.json
- Убедитесь, что файл существует и не поврежден
- Попробуйте использовать абсолютный путь

### Ошибки с цветовыми фильтрами
- Проверьте синтаксис JSON в config.json
- Убедитесь, что все значения в правильном формате (массивы из 3 чисел)
